## ⚡ Key Features

- Control welding machines with **Mitsubishi PLC**
- Supports multiple interface options: HMI Samkoon, HMI Kinco, Python GUI
- Real-time monitoring of welding parameters
- Modular and customizable for industrial use

---

## 🖥️ Interface Options

### 💻 HMI Samkoon

- Interactive touchscreen interface
- Full integration with Mitsubishi PLC
- Real-time welding parameter adjustment

### 🖲️ HMI Kinco

- Industrial-grade HMI alternative
- PLC-compatible
- Touchscreen interface

### 🐍 Python GUI

- Desktop GUI based on Python
- Cross-platform (Windows/Linux)
- Real-time status monitoring and manual control
